Features

🥕

歩数→食材ドロップ

歩けば歩くほど食材がドロップ。レア食材を求めてもっと歩こう。

🍳

レシピコレクション

ハンバーグ、パスタ、スープ…家庭料理が次々と完成。コレクションを埋めよう。

🎯

目標設定

歩数・消費カロリー・時間から目標タイプを選択。達成するとドロップ率がアップ。

How it Works

1

Apple Watch を装着して歩く

バックグラウンドで歩数を自動カウント。コンプリケーションで進捗を確認。

2

食材がドロップされる

一定歩数ごとに食材をゲット。Watch に通知が届いたら新しい食材の合図。

3

料理が完成！コレクションに追加

レシピの食材が揃ったら料理が完成。iPhone でコレクションを眺めよう。
